## Chromium

### Install

```
sudo pacman -S chromium
```

### Configurate

Open chrome://settings/

* Search engines

* Password Manager

Disable "Offer to save passwords"

* Payment methods

Disable "Save and fill payment methods"

* Addresses and more

Disable "Save and fill addresses"

* Google Translate

Disable "Use Google Translate"
